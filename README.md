# Fake News Detection using Machine Learning

This project detects whether a news article is FAKE or REAL using Natural Language Processing (NLP) techniques.

## Features
- Text preprocessing (cleaning, lowercasing)
- TF-IDF vectorization
- Machine Learning model for classification
- User input prediction system

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Dataset
Dataset includes:
- Fake news dataset
- True news dataset

## How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Run the script:
   python test.py

3. Enter custom news text to test

## Example
Input: "Narendra Modi is the Prime Minister of India"  
Output: REAL
